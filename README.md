## Descripción del Proyecto
esta aplicacion permite buscar peliculas y series usando la API de TMDB

## Características
- Visualización de películas y series populares
- Búsqueda por título
- Diseño responsivo con Bootstrap
- Página de detalle

## Tecnologías Utilizadas
- Python 3.11
- Flask
- Bootstrap 5
- Jinja2
- API TMDB
